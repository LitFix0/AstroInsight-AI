# AstroInsight-AI
This is a simple web application that takes user birth details (Name, Date, Time, Place) and provides a personalized astrology-based reading using the Google Gemini large language model (LLM). It consists of a Python Flask backend and a static HTML/JavaScript frontend.
Follow these steps to get the AI Astrologer running on your local machine.

Prerequisites
Before you begin, ensure you have the following installed and configured:

Python 3.9+: Download the latest version from python.org.

pip: Python's package installer, which typically comes bundled with Python,Google Cloud Project & Gemini API Key:You'll need a Google Cloud Project to access the Gemini API. 

1. Clone the Repository
2. Install Dependencies(pip install Flask Flask-Cors google-genai)
3. API key is directly included in the code
4. use python app.py to run the code
